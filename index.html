Audit Dog
Immutable Repository of Software Security Audits

About

When you use cryptocurrencies or perform any sensitive work on your computer, you need to rely on the security of the software you're using. In the fast-evolving landscape of cryptocurrencies, it's hard to know which piece of software (say, a wallet, mining GUI or a library) is safe to use. Audit Dog is an immutable database where community members can place their opinions on the safety of software so that it's easy for users to find an review.
